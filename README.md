Two types of Export/Import
1. Default Export/Import
   Ex. export default component_name  ->  export
   Ex. import component_name from "path"  ->  import
2. Named Export/Import
   Ex. export { component_name }  ->  export
   Ex. import { component_name } from "path"  ->  import
